# Casablanca Popcorn static site

This is a plain static website: no WordPress, database, plugins, or build process required.

## Publish

Upload the contents of this folder to the web root for `casablancapopcorn.com` (usually named `public_html`, `www`, or `htdocs`). Make a backup of the current WordPress site first, and arrange a redirect if the old site uses any URLs beyond the home page.

## Brand fonts

The stylesheet is set up for **AM Udine** (display), **Myriad Variable Concept Light Semi-Extended** (interface), and **Niramit** (supporting copy). Send the Adobe Fonts kit URL and add it to `index.html`; Niramit already loads from Google Fonts as the fallback.
